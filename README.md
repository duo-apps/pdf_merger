# PDF Merger
Merge multiple PDF files into one. 

# Usage

```
% python pdf_merger.py -h
usage: pdf_merger.py [-h] [-o OUTPUT] [-f [FILES ...]]

optional arguments:
  -h, --help            show this help message and exit
  -o OUTPUT, --output OUTPUT
                        Output PDF file.
  -f [FILES ...], --files [FILES ...]
                        PDF files to merge.
```
